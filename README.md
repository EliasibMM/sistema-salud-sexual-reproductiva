# Sistema Informativo de Salud Sexual y Reproductiva

Proyecto académico orientado a mejorar el conocimiento de los jóvenes sobre salud sexual y reproductiva mediante un sistema interactivo que evalúa conocimientos y genera recomendaciones educativas personalizadas.

## Funcionalidades

- Información sobre métodos anticonceptivos
- Información sobre infecciones de transmisión sexual
- Test interactivo de conocimientos
- Análisis inteligente de resultados
- Generación de plan educativo personalizado

## Tecnologías utilizadas

- Java
- Interfaz gráfica Java Swing
- Metodología ágil Scrum
- Trello para gestión del proyecto

## MVP

El MVP consiste en un sistema funcional que permita:

1. Realizar un test sobre salud sexual
2. Analizar las respuestas
3. Generar recomendaciones educativas
